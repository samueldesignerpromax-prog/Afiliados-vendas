# 🎓 Plataforma de Cursos com Sistema de Afiliados

Uma plataforma completa para venda de cursos online com sistema de afiliados integrado e chatbot com IA.

## 🚀 Funcionalidades

- ✅ Catálogo de cursos
- ✅ Chatbot com IA para atendimento
- ✅ Sistema de compra integrado
- ✅ Programa de afiliados com comissões
- ✅ Registro automático de vendas
- ✅ Liberação automática de acesso
- ✅ Dashboard para afiliados

## 🛠️ Tecnologias

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **IA**: Chatbot com respostas inteligentes
- **Deploy**: Vercel
